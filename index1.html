Selektory CSS

Stosowanie CSS do różnych części HTML

1. Selektor elementów wybiera elementy na podstawie nazwy znacznika HTML.

  1. Selektory klas wybierają elementy na podstawie wartości atrybutu klasy.
  2. Selektory ID wybierają elementy na podstawie wartości atrybutu identyfikatora.
  3. Selektory atrybutów wybierają elementy na podstawie ich atrybutów i wartości.
  4. Uniwersalny selektor wybiera we wszystkie elementy.
